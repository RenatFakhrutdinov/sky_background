The sky is beautiful! Now you can transfer a piece of the sky to your application using this library üëç

## How to use

Just wrap your widget in SkyBackground

```
import 'package:sky_background/sky_background.dart';

  Widget build(BuildContext context) {
    return Scaffold(
      body: SkyBackground(
        child: Center(
          child: Text('Widget'),
        ),
      ),
    );
  }

```
